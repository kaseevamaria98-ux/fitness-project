# Система записи на спортивные тренировки "Fit Arena"

Ссылка на репозиторий: https://github.com/kaseevamaria98-ux/fitness-project

## 1. Краткое описание проекта

### 1.1 Ф.И.О. разработчиков
Кащеева Мария Алексеевна

### 1.2 Группа
ДЭ-306

### 1.2 Тема проекта
Проектирование и разработка информационной системы для онлайн-регистрации клиентов на групповые тренировки в фитнес-центре "Fit Arena"

### 1.4 Цель проекта
Разработать информационную систему для автоматизации процесса записи клиентов на групповые тренировки в фитнес-центре "Fit Arena", включая проектирование баз данных, интерфейсов и бизнес-процессов.

### 1.5 Задачи проекта
1. Провести анализ предметной области "Фитнес-центр"
2. Разработать модели бизнес-процессов в нотациях IDEF0, BPMN, DFD
3. Спроектировать структуру базы данных и создать ER-диаграмму
4. Разработать UML-диаграммы системы (диаграммы классов, прецедентов, последовательности)
5. Создать ментальную карту системы
6. Оформить полную документацию проекта в формате Markdown

### 1.6 Аннотация
Проект представляет собой комплексное проектирование информационной системы для фитнес-центра "Fit Arena". Система позволяет клиентам осуществлять онлайн-запись на тренировки, тренерам — управлять расписанием занятий, администраторам — контролировать посещаемость и формировать отчетность. В проекте реализованы все этапы проектирования: от анализа бизнес-процессов до проектирования базы данных и интерфейсов.
