# Система записи на спортивные тренировки "Fit Arena"

Ссылка на репозиторий: https://github.com/kaseevamaria98-ux/fitness-project

## 1. Краткое описание проекта

### 1.1 Ф.И.О. разработчиков
Кащеева Мария Алексеевна

### 1.2 Группа
ДЭ-306

### 1.3 Тема проекта
Информационная система для онлайн-записи клиентов на групповые тренировки в фитнес-центре

### 1.4 Цель проекта
Создать удобную систему, которая позволяет клиентам записываться на тренировки, а администраторам — управлять расписанием и клиентами.

### 1.5 Задачи проекта
1. Изучить, как работает запись на тренировки
2. Нарисовать схемы процессов (IDEF0, BPMN)
3. Спроектировать базу данных
4. Создать диаграммы системы (UML, ERD)
5. Оформить всё в виде документации

### 1.6 Аннотация
Это учебный проект, который показывает, как можно автоматизировать запись на фитнес-тренировки. Включает схемы, диаграммы и описание работы системы.
